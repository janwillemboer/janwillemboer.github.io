If statements spiekbrief

int leeftijd = 10;

// if
//  conditie tussen ()
//   vergelijk twee waarden in de conditie
//                 uit te voeren code tussen { }
if (leeftijd < 18) {
    SaxionApp.printLine("Je mag nog niet stemmen!");
}

// ifs mogen ook in elkaar "genest" worden:
if (leeftijd < 18) {
    SaxionApp.printLine("Geen 18, geen druppel!");

    if (leeftijd < 12) {
        SaxionApp.printLine("En geef die fles onmiddellijk hier!!");
    }
}

// getallen vergelijken
// < kleiner dan
// > groter dan
// <= kleiner dan of gelijk aan
// >= groter dan of gelijk aan
// == gelijk 
// != niet gelijk 

// Strings vergelijk je met equals of equalsIgnoreCase:
String geslacht = "mannelijk";
if (geslacht.equals("vrouwelijk")) {
    SaxionApp.printLine("Dag mevrouw");
}

// code die uitgevoerd moet worden in de andere gevallen
// zet je in een else if of else block:
String aanhef;
if (geslacht.equalsIgnoreCase("vrouwelijk")) {
    aanhef = "Dag mevrouw";
} else if (geslacht.equalsIgnoreCase("mannelijk")) {
    aanhef = "Dag meneer";
} else {
    aanhef = "Dag mens";
}

// een switch-statement laat je verschillende gevallen vergelijken
int getal = 5;
// switch
//   tussen haakjes de te vergelijken waarde
//        accolades
switch (getal) {
    // case 
    //   waarmee je de waarde vergelijkt
    //    dubbele punt
    //        en dan zonder accolades de uit te voeren code,
    //        die je afsluit met break;
    case 1:
        SaxionApp.printLine("De waarde is 1");
        break;
    case 2:
        SaxionApp.printLine("De waarde is 2");
        break;
    default: // voor alle andere gevallen:
        SaxionApp.printLine("Ik weet het ook niet meer");
        break;
}

Extra oefeningen if-statements

  1. Laat het programma een willekeurig getal kiezen uit de reeks 1, 2 of 3. Laat de gebruiker het getal raden. Geef aan of hij het goed of fout heeft geraden.
  2. Laat de gebruiker een maand invoeren. Het programma laat zien welk maandnummer deze maand heeft. Het programma geeft een vriendelijke foutmelding als het de maand niet herkent.
  3. Vraag de gebruiker om een getal tussen 1 en 10. Check met het programma of het getal inderdaad binnen die reeks valt. Als dat niet zo is, straf de gebruiker dan op vreselijke wijze.
  4. Doe opnieuw de opdracht 2 van "methoden met returnwaarden" hierboven. Check nu of de tekst inderdaad 8 tekens lang is. Voor het bepalen van de lengte van een String gebruik je de methode .length().
  5. Vraag de gebruiker om zijn naam, leeftijd en geslacht. Bepaal op basis van leeftijd en geslacht wat de beste manier is om deze persoon aan te spreken, en begroet deze persoon op deze manier.
  6. Kies twee willekeurige getallen. Vraag aan de gebruiker om deze op te tellen. Laat de gebruiker weten of het antwoord klopt.
  7. Vraag de gebruiker om een woord in te voeren. Check of dit woord de letters "ch" bevat en toon aan de gebruiker of dat wel of niet zo is. Met String.contains() kun je checken of een string een andere string bevat.
