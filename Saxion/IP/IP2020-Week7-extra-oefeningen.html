Spiekbrief Loops 3 (de gewone for-loop)

// Een for-loop gebruik je als je een teller nodig hebt. 
// for, haakje openen
// |  initialisatie van de teller (in dit geval: "i")
// |    |  puntkomma
// |    |       conditie om de lus te herhalen
// |    |        |   puntkomma
// |    |        |      ophogen van de teller, haakje sluiten
// |    |        |       |    body van de loop tussen { }
// |    |        |       |     |
for (int i = 0; i < 10; i++) {
    SaxionApp.printLine("De teller is nu: " + i);
}

// dit is exact hetzelfde als een lus die je eerder hebt geleerd:
int i = 0;
while (i < 10) {
    SaxionApp.printLine("De teller is nu: " + i);
    i++;
}

// for loops mogen uiteraard ook "genest" worden. 
// dit doe je bijvoorbeeld als je een kolommen en rijen nodig hebt.
// Voorbeeld van een geneste for-loop met kolommen en rijen:
for (int rij = 0; rij < 10; rij++) {
    for (int kolom = 0; kolom < 20; kolom++) {
        SaxionApp.print(" x");
    }
    SaxionApp.printLine();
}

// for loops kunnen ook de andere kant op tellen
for (int i = 10; i > 0; i--) {
    SaxionApp.printLine("De terugtellende teller is nu: " + i);
}

// for loops kunnen ook grotere stappen nemen. Hier een for-loop die elke keer de teller met 2 ophoogt:
for (int i = 0; i < 10; i = i + 2) {
    SaxionApp.printLine("De teller is nu: " + i);
}

Spiekbrief Lists 2 (de gewone array)

// Net als een ArrayList bevat een array een lijst met waarden.
// Een array is sneller dan een ArrayList, maar veel primitiever.

// declareren:
// type, met blokhaken erachter om aan te geven dat het een array is
// |  variabelenaam
// |   |
int[] getallen;

// initialiseren:
// variabelenaam
// |    isgelijkteken
// |     |  new
// |     |   |   type 
// |     |   |   | blokhaken met de grootte van de array
// |     |   |   |  |
getallen = new int[10];

// declareren en intialiseren kan ook tegelijk:
String[] namen = new String[10];

// waarden in de array zetten. De index gaat van 0 tot Array.length - 1,
// vergelijkbaar met een ArrayList
getallen[0] = 10;
getallen[1] = 20;

// een array declareren, initialiseren en gelijk ook met waarden vullen:
int jaartallen = {1980, 1990, 2000, 2010, 2020};

// waarden opvragen uit de array
SaxionApp.printLine(getallen[0]);
SaxionApp.printLine(getallen[1]);

// over een array heen lussen
for (int i = 0; i < namen.length; i++) {
    SaxionApp.printLine(namen[i]);
}

// of met de enhanced for loop:
for (String naam : namen) {
    SaxionApp.printLine(naam);
}

// alle typen kunnen als array gebruikt worden, ook eigen typen:
double[] waarden = new double[4];
waarden[3] = 3.14;

Circle[] circles = new Circle[10];
circle[0] = new Circle();
circle[0].radius = 10;

Extra oefeningen Lists 2

  1. Maak een array van Strings met 5 elementen. Laat de gebruiker deze 5 elementen invullen met verschillende woorden. Druk ze daarna op het scherm af.
  2. Neem de array van de vorige oefening, en druk nu het eerste en het laatste element af. Als je een foutmelding krijgt, is er iets mis met de index, zie de spiekbrief.
  3. Neem de lijst van de vorige oefening, maar druk deze nu op lengte af. Dus het kortste woord eerst en het langste woord het laatst. Verzin of zoek een handig sorteeralgoritme, of gebruik de volgende tip: druk eerst alle woorden met lengte 1 af, daarna met lengte 2, enz., tot en met de lengte van het langste woord.
  4. Maak een nieuwe klasse met de naam Positie, of iets dergelijks, die als eigenschappen een x en een y coördinaat heeft. Maak een array die vier Positie objecten kan bevatten. Vul de vier posities met vier verschillende coördinaten: laat dit eventueel door de gebruiker invullen. Laat het programma vervolgens alle coördinaten op het scherm tonen en met lijnen met elkaar verbinden. Verbind ook de laatste en eerste coördinaat met elkaar, zodat je een vierhoek op het scherm krijgt.
  5. Maak een array van vier integers en vul ze met een paar waarden. Maak een methode om twee elementen uit de array om te wisselen. Wissel twee elementen om en druk de lijst af om te controleren of het werkt.
  6. BONUS: Als je een manier hebt om twee elementen van een array van plek te laten wisselen, dan kun je daarmee een array ook sorteren. Vergelijk elk element van de array met het volgende element in de array. Als het element groter is dan het volgende element, wissel de twee elementen dan om. Herhaal dit hele verhaal totdat je niks meer hebt omgewisseld: dan weet je dat de lijst goed is gesorteerd.
