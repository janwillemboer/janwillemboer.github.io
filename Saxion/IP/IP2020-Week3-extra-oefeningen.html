Loops spiekbrief

String antwoord = "ja";
// while
//    tussen haakjes de conditie
//    (vaak vergelijk je twee waarden, net als bij een if)
//                 In de accolades de body die herhaald moet worden
//                 De body wordt nooit uitgevoerd als de conditie nooit waar is.
while (antwoord.equals("ja")) {
    SaxionApp.printLine("Doorgaan?");
    antwoord = SaxionApp.readString();
}

// do
//   in de accolades de body die herhaald moet worden
//   De body wordt altijd minstens 1x uitgevoerd, 
//   omdat de conditie pas op het eind wordt gechecked
// na de accolades: while 
//       met tussen haakjes weer de conditie
//                    en puntkomma achter de haakjes!
String herhalen;
do {
    SaxionApp.printLine("Nog een keer herhalen?");
    herhalen = SaxionApp.readString();
} while (herhalen = "ja");

// een while lus met een teller, die van 1 tot 10 telt
int teller = 1;
int tellenTotEnMet = 10;
while (teller <= tellenTotEnMet) {
    SaxionApp.printLine(teller);
    teller++;
}

// while lussen kunnen ook weer genest worden
// dit fragmentje print een matrix van 20 keer 10 X-jes.
int maxAantalRijen = 10;
int maxAantalKolommen = 20;
int rij = 1;
while (rij <= maxAantalRijen) {
    int kolom = 1;
    while (kolom < maxAantalKolommen) {
        SaxionApp.print(" X ");
        kolom++;
    }
    SaxionApp.printLine("");

    rij++;
}

// eindeloze lus
while (true) {
    SaxionApp.printLine("Dit stopt nooit");
}

// eindeloze lus die toch een keer stopt
int getal = 4;
while (true) {
    // (doe dingen)
    if (getal > 10) {
        break; // stopt de lus
    }
}

If statements 2 spiekbrief

int leeftijd = 17;
// ken condities toe aan booleans
// boolean
//      variabelenaam
//            isgelijkteken
//                conditie zonder haakjes
//                           puntkomma
boolean isTeOud = leeftijd > 88;
boolean isTeJong = leeftijd < 8;

// booleans kun je gebruik in if-statements
if (isTeOud) {
    SaxionApp.printLine("Je bent te oud.");
}
if (isTeJong) {
    SaxionApp.printLine("Je bent te jong");
}

// je kunt booleans en condities combineren met de volgende operators:
// && = AND, beide condities moeten true zijn 
// || = OR, één van beide condities moet true zijn
// ! = NOT, keer de conditie om die erachter staat.
if (isTeOud || isTeJong) {
    SaxionApp.printLine("Sorry, je mag niet meedoen, je bent te oud of te jong.");
}

// ook String vergelijkingen kun je op die manier gebruiken
String gerecht = "Gebraden varkensvlees";
SaxionApp.printLine("Bent u vegetariër?");
String antwoord = SaxionApp.readString();
boolean isVegetarier = antwoord.equals("ja") || antwoord.equals("j");
if (isVegetarier && !gerecht.contains("vlees")) {
    SaxionApp.printLine("Het gerecht voldoet aan uw eisen, er zit geen vlees in");
}

// je kunt dit alles ook op andere plekken gebruiken
// waar je condities gebruikt:
while (isTeOud || isTeJong) {
    // enz.
}

Extra oefeningen If statements 2

  1. Probeer opnieuw je huisgenoten en/of familie te identificeren. Vraag een serie persoonlijke vragen aan de gebruiker, die deze met ja of nee mag antwoorden. Sla de antwoorden op in boolean variabelen. Combineer daarna de antwoorden in verschillende condities om te bepalen wie er achter de computer zit.
  2. Laat de gebruiker twee verschillende zinnetjes invoeren. Controleer of het echte zinnen zijn door
    • te checken of er een spatie in staat,
    • en door te checken of de lengte groter is dan 10 tekens óf dat er een punt aan het eind van de zin staat. Tip: gebruik String.endsWith().
    • De zinnen mogen niet aan elkaar gelijk zijn.
  3. Laat de gebruiker zijn leeftijd invoeren. Laat het programma daarna twee willekeurige getallen verzinnen die de gebruiker bij elkaar moet optellen. Als de gebruiker jonger is dan 10, dan mag hij drie pogingen doen om het goede antwoord te verzinnen. Als hij jonger is dan 17, dan mag hij twee pogingen doen. Alle anderen mogen maar 1 poging doen.
  4. Maak een programma dat een vierkantje in het midden van het SaxionApp scherm tekent. We gaan deze weer animeren, maar nu zorgen we ervoor dat je het blokje kunt besturen door W, A, S of D in te drukken. Gebruik SaxionApp.readChar() om te wachten op een toets. Zorg ervoor dat het blokje naar boven gaat als je W indruk, naar links als je A indrukt, naar beneden als je S indrukt, en naar rechts als je D indrukt. Als het blokje de randen raakt, mag hij niet verder naar die kant.
