Spiekbrief Lists 1

// lijst maken 
// ArrayList
//      type tussen < en >
//             variabelenaam
//                   isgelijkteken
//                         new ArrayList<>()
//                                    puntkomma
ArrayList<String> namen = new ArrayList<>();

// let op dat je dat voor ints e.d. net iets anders doet,
// omdat die niet als type tussen de < en > mogen staan.
// int wordt Integer
// char wordt Character
// double wordt Double
// boolean wordt Boolean
ArrayList<Integer> nummers = new ArrayList<>();

// iets toevoegen aan een lijst: .add()
namen.add("Kees");
nummers.add(10);

// de grootte van de lijst opvragen:
int grootte = namen.size();

// het eerste en tweede element van een lijst opvragen
// Let op, de index begint bij 0 te tellen
String naam = namen.get(0);
String andereNaam = namen.get(1);

// checken of een lijst een element bevat
if (namen.contains("Kees")) {
    SaxionApp.printLine("Kees staat op de lijst");
}

Spiekbrief Loops 2

ArrayList<String> namen = new ArrayList<>();

// iets doen met elk element van een ArrayList:
int index = 0;
while (index < namen.size()) {
    String naam = namen.get(index);
    SaxionApp.printLine(naam);
    index++;
}

// datzelfde met de enhanced for loop
// for
//   tussen haakjes:
//     variabele om elk
//     element in te ontvangen
//             dubbele punt
//                 de lijst
// loop body tussen accolades
for (String naam : namen) {
    SaxionApp.printLine(naam);
}

Extra oefeningen Lists 1 en Loops 2

  1. Laat de gebruiker een paar getallen ingeven, totdat de gebruiker het getal 0 invult. Maak twee verschillende lijsten met deze getallen: een lijst met de getallen onder nul, en een lijst met de getallen boven nul. Druk beide lijsten af.
  2. Laat de gebruiker een lijst met woorden vullen, totdat de gebruiker het woord leeglaat. Toon daarna elk woord op het scherm, behalve de woorden waar de letter "z" in voorkomt.
  3. Laat de computer een lijst van 5 willekeurige getallen tussen 1 en 10 maken. Er mogen niet twee dezelfde getallen op de lijst staan. Druk de lijst af op het scherm.
  4. Neem de lijst van de vorige opdracht, maar druk ze nu niet af op het scherm. Je programma moet de gebruiker vragen de getallen allemaal te raden in maximaal 10 pogingen. Als de gebruiker een getal raadt, dan moet de computer die van de lijst verwijderen. Als de gebruiker de getallen allemaal raadt, heeft hij gewonnen. Als hij ze binnen 10 pogingen nog niet heeft geraden, heeft hij verloren.
  5. Laat de gebruiker een lijst met woorden vullen, totdat de gebruiker het woord leeglaat. Maak een methode die een ArrayList als parameter heeft, en die een nieuwe ArrayList als returnwaarde heeft. Noem de methode "filter". Zorg ervoor dat de methode de lijst filtert zodat alleen woorden met een "e" erin overblijven, en geef dat als returnwaarde terug. Test of de methode werkt.
  6. Laat de gebruiker een paar waarden invoeren tussen de 1 en 100. Teken nu met de SaxionApp deze waarden als een grafiek op het scherm. Tip: gebruik de index * 10 als de x-as en gebruik de waarde als de y-as.
  7. Laat de gebruiker een paar waarden invoeren tussen de 1 en 100. Maak een programma dat de lijst daarna sorteert en op het scherm toont. Je moet dit sorteren niet zelf gaan maken (mag wel, maar dat is best lastig). Zoek op het internet naar een makkelijke manier om de lijst te sorteren.
