Variabelen declareren en intialiseren spiekbrief

// type
//   naam
//      isgelijkteken
//           initiële waarde
//             statement afsluiten met puntkomma
int getal = 10;         // int is een geheel getal (integer)
double pi = 3.14;    // double is een kommagetal
String tekst = "Hoi";   // String is een tekst
boolean isHetZo = true; // boolean is true of false
char letter = 'y';      // char is 1 teken

// je mag het declareren en initialiseren splitsen:
int nummer;  // declareren
nummer = 11; // initialiseren

// gebruik variabelen om methoden mee aan te roepen
SaxionApp.printLine(tekst);
SaxionApp.printLine(getal);

// variabelen kun je veranderen in iets anders
// nu niet meer het type ervoor, maar gelijk de naam
//   isgelijkteken
//       nieuwe waarde
getal = 11;
tekst = "Hallo";
isHetZo = false;

// gebruik variabelen om mee te rekenen
// + is optellen
// - is aftrekken
// * is vermenigvuldigen
// / is delen
// % is modulo, dat is de restwaarde van een deling (10 % 6 = 4)
int optelling = getal + getal;
double omtrek = pi * 2;
double deling = omtrek / 10;

// let op dat je bij delen altijd doubles gebruikt.
// als je integers gebruikt, dan wordt er naar beneden afgerond.
// om dit te voorkomen moet je integers "casten" naar double:
deling = (double)getal / 10;

// tel 1 bij een variabele op
getal = getal + 1; // getal wordt nu 1 meer
// omdat dit zo vaak voorkomt, is daar een kortere versie van bedacht:
getal++;          // getal wordt nu 1 meer.

// combineer teksten
String naam = "Kees";
String begroeting = "Hallo " + naam;

// combineer teksten en getallen
tekst = "De omtrek is " + omtrek;

// mag ook gelijk in de methode-aanroep
SaxionApp.printLine("Hallo " + naam + ", de omtrek is " + omtrek);

Extra oefeningen variabelen

  1. Maak twee variabelen met hele getallen (integers). Zorg ervoor dat je programma de volgende dingen uitrekent en netjes laat zien:
    • tel ze bij elkaar op
    • trek ze van elkaar af
  2. Maak twee variabelen met hele getallen (integers). Zorg ervoor dat je programma de volgende dingen uitrekent en netjes laat zien:
    • vermenigvuldig ze met elkaar
    • reken de restwaarde uit van de deling
  3. Maak twee variabelen met getallen met decimalen (doubles). Zorg ervoor dat je programma de volgende dingen uitrekent en netjes laat zien:
    • vermenigvuldig ze met elkaar
    • deel de getallen door elkaar
  4. Maak twee variabelen met getallen. Zorg ervoor dat je programma het percentage berekent dat het ene getal van het andere is.
  5. Maak twee integer variabelen, de een met de waarde 2 en de ander met de waarde 3. Zorg ervoor dat je programma de eerste door de tweede deelt en de juiste uitkomst geeft, namelijk iets in de buurt van 0.66666. Als je programma niet de juiste waarde oplevert, kijk dan naar wat "casten" is hierboven.
  6. Maak een programma waarin je deze variabelen gebruikt: pi en straal. Gebruik voor pi de waarde 3.14159. Zorg er nu voor dat het programma met een nette melding afdrukt wat de omtrek en de oppervlakte van een cirkel met de opgegeven straal is (gebruik voor straal een waarde die je zelf verzint). De berekening voor de omtrek is 2 keer pi keer de straal. De berekening voor de oppervlakte is pi keer straal keer straal.
  7. Maak een variabele voor een begroeting en een voor een naam. Laat je programma deze achter elkaar op het scherm afdrukken door meerdere keren SaxionApp.print() aan te roepen.
  8. Hetzelfde als de vorige, maar gebruik nu één keer SaxionApp.printLine() om hetzelfde te bereiken, en combineer in de aanroep de variabelen met elkaar.
  9. Schrijf een verhaaltje van een paar zinnen over een alledaagse situatie, waarbij een paar woorden in variabelen staan. Toon dit verhaal op het scherm. Verzin nu andere waarden voor de variabelen, waardoor het verhaal in een horrorstory verandert.
