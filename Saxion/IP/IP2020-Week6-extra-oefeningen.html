IP 2020 - Extra oefeningen week 6 (Variabelen 2)

Voor het doen van extra oefeningen gebruik je het Sandbox project, dat je van Blackboard kunt downloaden bij Week 1.1. Dit is een leeg project waar je zelf code in kunt schrijven voor de SaxionApp.

Spiekbrief variabelen 2 (klasses)

// een eigen klasse definiëren:
// public
// |   class
// |    |    naam van de klasse
// |    |      |            eigenschappen 
// |    |      |            tussen accolades
public class MijnPrachtigeKlasse {

    // eigenschappen van de klasse:
    // public
    // |   type
    // |    |     naam
    // |    |       |  puntkomma
    public String titel;
    public int getal;
}

// dit moet in een losse java-file staan, die dezelfde naam 
// heeft als de naam van de klasse, dus in dit geval 
// MijnPrachtigeKlasse.java

// eigen klasse als variabele gebruiken:
// type (= naam van de klasse)
//  |          naam v/d variabele
//  |               |   isgelijkteken
//  |               |    |  new + naam van de klasse
//  |               |    |  |      |         lege haakjes
//  |               |    |  |      |               | puntkomma
MijnPrachtigeKlasse ding = new MijnPrachtigeKlasse();

// Dit heet de klasse *instantiëren*, zodat je een *object* 
// van die klasse krijgt.

// eigenschappen zetten gaat net als variabelen:
ding.titel = "Mijn titel";
ding.getal = 10;

// eigenschappen opvragen ook:
SaxionApp.printLine("De titel is: " + ding.titel);
int som = ding.getal + 20;

// andere objecten van dezelfde klasse hebben hun eigen data
// zonder dat dit de data van eerder gemaakte objecten beïnvloedt
MijnPrachtigeKlasse anderDing = new MijnPrachtigeKlasse();
anderDing.title = "Andere titel";
anderDing.getal = 123;

SaxionApp.printLine(anderDing.titel); // "Andere titel"
SaxionApp.printLine(ding.titel); // "Mijn titel"

// een eigen klasse als parameter van een method gebruiken:
public void drukInfoAf(MijnPrachtigeKlasse x) {
    SaxionApp.printLine(x.titel + " = " + x.getal);
}

// een eigen klasse als returnwaarde van een method gebruiken
public MijnPrachtigeKlasse maakDing() {
    MijnPrachtigeKlasse x = new MijnPrachtigeKlasse();
    x.titel = "automatisch gegenereerd";
    return x;
}

// als een klasse het returntype is, kun je "null" gebruiken
// om aan te geven dat er een lege waarde gereturned wordt:
public MijnPrachtigeKlasse leeg() {
    return null;
}

// een eigen klasse in een arraylist gebruiken
ArrayList<MijnPrachtigeKlasse> lijst = new ArrayList<>();

MijnPrachtigeKlasse x = new MijnPrachtigeKlasse();
lijst.add(x);

Extra oefeningen variabelen 2

  1. Maak een nieuwe klasse "Cursus", met een eigenschap "naam" en een eigenschap "waardering". Maak nu in je programma (minstens) twee verschillende variabelen van het type Cursus om twee verschillende cursussen mee op te slaan. Gebruik dit om de cursussen waar je nu aan deelneemt als titel, en geef bij waardering aan wat je van de cursus vindt. Druk deze gegevens nu een voor een op het scherm af.
  2. Maak een nieuwe klasse Cirkel, waar je de x-coördinaat, de y-coördinaat en de straal van een cirkel in kunt opslaan. Laat de gebruiker een x-coördinaat, een y-coördinaat en een straal invoeren. Sla deze gegevens op in een nieuw object van de klasse die je zojuist hebt gemaakt. Maak daarna een methode die als parameter een Cirkel meekrijgt, en die de cirkel op het scherm tekent.
  3. Pas de vorige oefening aan, zodat de gebruiker drie cirkels kan invoeren. Sla de cirkel objecten op in verschillende variabelen. Gebruik je methode uit de vorige oefening om de cirkels op het scherm te tonen.
  4. Pas de vorige oefening aan, zodat de gebruiker niet één of drie, maar oneindig veel cirkels kan invoeren totdat hij als straal "0" invoert. Sla de cirkel objecten op in een lijst en gebruik deze daarna om alle cirkels op het scherm te tonen.
  5. Maak een nieuwe klasse Woord, die bedoeld is om een woord in een wordcloud te tonen. De klasse moet de eigenschappen "tekst", "grootte", x, y, en "kleur" hebben. Tip: gebruik java.awt.Color als type voor de kleur. Zet hiervoor import java.awt.*; bovenin het bestand bij de andere imports, nu kun je Color gebruiken. Vul een lijst met verschillende van deze woorden. Maak daarna een methode die de lijst afdrukt als wordcloud.
  6. In Minecraft kun je twee items die versleten zijn combineren tot een nieuw item. Dit gaan we hier simuleren.
    • Maak een klasse "MinecraftItem", die een omschrijving heeft en een eigenschap "health". Health krijgt straks een waarde van 0 tot 100: 0 is totaal versleten, 100 is nog helemaal goed.
    • Maak nu drie objecten in drie verschillende variabelen: twee zwaarden die 80% en 20% health hebben, en een schoffel die op 90% zit.
    • Maak een methode "combine", die twee MinecraftItems als parameters meekrijgt, en die een MinecraftItem als returnwaarde heeft. De methode checkt of de items dezelfde omschrijving hebben en returnt een nieuw item met 100% health als dat zo is. De methode returnt een item met omschrijving "FOUT" met 0% health als dat niet zo is.
    • Probeer in je programma de werking van je klasse en methode uit. Druk de items af op het scherm, combineer ze, en druk af wat er terugkomt uit de methode.
